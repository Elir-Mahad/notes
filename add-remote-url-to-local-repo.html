How to add a remote url to your local project folder

If you have a folder on your computer with some files, and you want to upload it to Github, then you must utilize something that's called the 'remote url'. A remote url is the url that connects your local folder to the repository that you have created on github. In this tutorial, i will show you how to use the remote url to establish this connection.

Important things: In this post, we will be using the terminal commands . To follow along, you should already be logged into visual studio code with your github account. In the code snippets, the green code is the terminal response, elir is my computers username, and Elir-Mahad is my github username. When you are trying out the code, elir will be replaced by your computers username, and Elir-Mahad will be replaced by your github username.

1. Create the project folder and files on your desktop.

  • For this tutorial, the name of the project folder will be 'game'.
  • Open your terminal.
  • Navigate into your Desktop folder.

    • cd Desktop

  • Confirm that you are in the Desktop folder.

    • pwd
    /Users/elir/Desktop

  • Create a folder called 'game'.

    • mkdir game

  • Navigate into the game folder.

    • cd game

  • Confirm that you are in the game folder.

    • pwd
    game

  • Create the file index.html inside the game folder.

    • touch index.html

  • Create the file styles.css inside the game folder.

    • touch styles.css

  • List all items in the game folder.

    • ls
    index.html styles.css

    2. Open the project folder inside visual studio code.

  • Open visual studio code.
  • In the top left corner, click on the explorer tab (right above the magnifying glass) .
  • Click on the open folder button.
  • Navigate to your folder.
  • Click open.
  • Minimize the visual studio code program.

    • 3. Create the github repository and copy its remote url.

  • Open your browser, go to github.com, and login.
  • In your github account, create a repository called game.
  • After you click on create repository, you will be automatically redirected to a Quick setup Page.
  • Inside the box that starts with "…or create a new repository on the command line", you will find the line:

    • git remote add origin https://github.com/Elir-Mahad/game.git

  • In your page, Elir-mahad will be replaced by your username.
  • Copy the git remote add origin line to your clipboard.

    • 4. Add the remote url of the github repository to your local folder.

  • Navigate back into your visual studio code.
  • Open the visual studio code terminal.
  • Verify that you are logged in with the correct github username.

    • git config --list
    user.name=elir-mahad
    user.email=mahad@gmail.com

  • In your terminal, elir-mahad should be replaced with your github username; and mahad@gmail.com should be replaced with your email.
  • If you are logged in, then initialize your folder.

    • git init
    Initialized empty Git repository in /Users/elir/Desktop/game/.git/

  • After initialzing your folder, paste the remote url in the visual studio code terminal.

    • git remote add origin https://github.com/Elir-Mahad/game.git

  • Confirm that your local folder is connected to your github repository.

    • git remote -v
    origin https://github.com/Elir-Mahad/game.git (fetch)
    origin https://github.com/Elir-Mahad/game.git (push)

  • This terminal response is evidence that the game folder on your desktop, is now connected to the game repository that's in your github.

    • 5. How to Remove an old remote url from a project and add a new one.

  • If you've already added a remote url to your project folder, but you want to replace it with a new remote url, then in your visual studio terminal type:

    • git remote remove origin

  • This should remove the previous remote url.
  • Verify that the previous url has been removed.

    • git remote -v

  • If there is no response, then the previous url has been removed.
  • If you already have a new remote url, then repeat step 4.
