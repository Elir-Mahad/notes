How to add a remote url to your local repository

If you have a folder on your computer with some files, and you want to upload it to Github, then you must utilize something that's called the 'remote url'. A remote url is the url that connects your local folder to the repository that you have created on github. In this post, i will show you how to use the remote url to establish this connection.

1. In your github account, create a repository:

2. Copy the remote origin url:

3. Open your visual studio code, then open your project inside it:

4. Open the visual studio code terminal and paste the remote url:

git remote add origin https://github.com/Elir-Mahad/game.git

4. Confirm that your local folder is connected to your github repository:

git remote -v

  • Terminal response should be:

    • origin https://github.com/Elir-Mahad/game.git (fetch)
    origin https://github.com/Elir-Mahad/game.git (push)

    5. Removing an old remote url from a project and adding a new one.

  • If you've already added a remote url to your folder, but you want to replace it with a new remote url then in your terminal type:

    • git remote remove origin
