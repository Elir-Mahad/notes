What are histograms ?

Histograms are useful for identifying the distribution of the data. In other words, are the data points equally distributed or are the data points clustered in one part of the graph. A histogram groups the number of data points based on the bin (which is a range), so you will see how the data points are distributed over the x-axis.

In the below example, althought both plots have the similar data, you can see that the data distribution is much more clear in the histogram:

How to make a histogram ?

import matplotlib.pyplot as plt
# import matplotlib
values = [0, 0.6, 1.4, 1.6, 2.2, 2.5, 2.6, 3.2, 3.5, 3.9, 4.2,6]
# the variable values contains a list of numbers
plt.hist(values, bins=3)
# make a histogram plot with all the values inserted in 3 bins
plt.show()
# show the histogram plot

Population pyramid with age, gender, and population.

The population pyramid below is made of two histograms. In the right histogram, the x-axis is the population in millions and the y-axis is the age range of women. In the left histogram, the x-axis is the population in millions, and the y-axis age range of the men. Individually, these histograms are fairly interesting. Additionally, by placing them next to each other, we're able to see the similarities and differences between the men and the women; such as the fact that, the largest population for both groups are in the age range of 40 to 44.

How to add axis labels and plot title.

import matplotlib.pyplot as plt
# import matplotlib
year = [1950, 1951, 1952… 2100]
# the variable year contains a list of years
pop = [2.538, 2.57, 2.62, ..., 10.85]
# the variable pop contains a list of populations
plt.plot(year, pop)
# make a plot
# x-axis values = year
#y-axis values= population

plt.xlabel('Year')
# add x-axis label
plt.ylabel('Population')
# add y-axis label
plt.title('World Population Projections')
# add plot title
plt.show()
# show the plot

How to add ticks (custom axis values) and historical data.

Ticks are custom labels for the axis values. For example: if the first value in your y-axis is 10, then you can use ticks to replace 10 with ‘10books’.

import matplotlib.pyplot as plt
# import matplotlib
year = [1950, 1951, 1952… 2100]
# the variable year contains a list of years
pop = [2.538, 2.57, 2.62, ..., 10.85]
# the variable pop contains a list of populations

year = [1800, 1850, 1900] + year
# add more data to the list years
pop = [1.0, 1.262, 1.650] + pop
# add more data to the list pop

plt.plot(year, pop)
# make a plot
# x-axis values = year, y-axis values= population
plt.xlabel('Year')
# add x-axis label
plt.ylabel('Population')
# add y axis label
plt.title('World Population Projections')
# add plot title

plt.yticks(
[0, 2, 4, 6, 8, 10],
['0', '2B', '4B', '6B', '8B', '10B']
) # add y-axis ticks
# the value 2 in the y-axis
will be replaced by 2B and so on
plt.show()
# show the plot
