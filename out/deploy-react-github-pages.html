How to deploy a react site on github pages

1. Connect your local folder to your repo and make your first commit.

2. Add github pages dependency.

yarn add gh-pages

3. Grab github pages url.

  • Go into the settings of your repository.
  • Go into the pages tab.
  • Change the branch to master branch & click save.
  • Refresh the page, and you will see your react apps url.
  • Copy and paste the url.

    • 4. Make edits to the package.json .

  • Open the 'package.json' file.
  • In the package.json, under [ private : true ] type:

    • 'homepage': 'https://elir-mahad.github.io/reactapp/',

  • In the package.json, in the scripts object, under [ eject : react-scripts-eject ] type:

    • 'predeploy': 'yarn build',
    'deploy': 'gh-pages -d build'

    5. Deploy the app.

  • In the laptop or vsc terminal:

    • git commit -m “Add Github Pages config.”
    yarn run deploy

  • In the github website:
  • go to settings, enter the branches tab, and change the branch from master to gh-pages branch.
  • Insert your react apps url into the browser. The react site should be live.

    • 6. To make changes to your app

  • After pushing your code, in the terminal type: yarn run deploy.
  • yarn run deploy will update the github pages the site with the new changes.
