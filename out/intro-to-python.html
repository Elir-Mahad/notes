Chapter 1

How to declare variables/constants in python ?

When declaring constants, you don’t have to preface the name of the constant with the world ‘var’ or ‘const’, instead you just write the word itself.

const height = 1.79 (javascript)
height = 1.79 (python)

Print plays a similar role as console log. Here is a javascript console log:

console.log(height)
1.79 // this is the output

Here is a python print statement:

print(height)
1.79 # this is the output

How many data types does Python have ?

These are some of the popular Python data types:

  • Text Type: str (string)
  • Numeric Types: int (integer solid number : 20), float (decimal number : 20.5)
  • Sequence Types: list, tuple, range
  • Mapping Type: dict
  • Boolean Type: bool

    • Adding different data types

    Adding different data types often leads to different behaviours.

    When you add integers they get added properly.

    2 + 3 = 5

    When you add strings they just get attached to each other.

    ‘Ab’ + ‘cd’ = ‘Abcd’

    Chapter 2

    What is a list ?

    In JavaScript, an array is a list of values.
    This is a javascript array : [1.73, 1.68, 1.71, 1.89]
    Each value is called an element and it’s identified through its index.
    So the element in index 2 is 1.71.

    In python, the word ‘list’ refers to the same construct as a javascript ‘array’.
    This is a python list : [1.73, 1.68, 1.71, 1.89]

    A python lists can contain different data types. For example:

    [ "liz", 1.73, "emma", 2, 1.68, "mom" , 1.71, 5, "dad" , 1.89]
    # the above list contains strings, integers, and floats

    How to select elements in a list ?

    In python, you can select list elements by using minus values. For example:

    fam = [ "liz", 1.73, "emma", 1.68, "mom" , 1.71, "dad" , 1.89]
    # the above variable fam stores a list with strings and numbers
    # in this list, the item on the left end has the index of 0 (“liz”)
    # the item on the right end has the index of -1 (1.89)
    # you can count up from 0 or you can count down from -1

    print(Fam[6])
    # print out the element in the 6th index
    dad
    # output

    print(Fam[-1])
    # print out the element in the index -1
    1.89
    # output

    Can Lists can contain other lists ?

    Yes. In python, lists (i.e, javascript arrays), can contain lists. For example:

    fam2 = [
    ["liz", 1.73],
    ["emma", 1.68],
    ["mom" , 1.71],
    ["dad", 1.89]
    ]
    # the above variable fam2 stores a list with strings and floats
    # the main list, contains 4 other lists

    Print (fam2[1,1])
    # print out the element in row 1, index 1 = 1.68

    How to select a range of elements in a list ?

    We can select a range of elements in a list through a process which is formally known as list slicing.
    This is the format for list slicing :
    [start of selected portion: end of the selected portion ]
    The starting index is included in the output, but the ending index is excluded from the output. For example:

    fam = [ "liz", 1.73, "emma", 1.68, "mom" , 1.71, "dad" , 1.89]
    # the above variable fam has a list with strings and floats

    print(fam[3:5])
    # print out the 3rd and 4th index: [1.68, ‘mom]
    # the 3rd index is printed out because according to the format it is supposed to be included in the output
    # the 5th index is not printed out because according to the format it is supposed to be excluded from the output

    print(fam[:4])
    # print out everything until 4th index : [ "liz", 1.73, "emma", 1.68 ]
    # since no starting index is provided, python starts counting from 0
    # the 4th index is not printed out because according to the format it is supposed to be excluded from the output

    print(fam[5:])
    # print out everything after the 5th index: [1.71, "dad" , 1.89]
    # the 5th index is the starting point; according to the format it is supposed to be included in the output
    # Since no ending index is provided, python counts to the end of the list

    How to change the value of a list element ?

    In the below example, we are going to change the value of a list element by redefining it:

    fam = [ "liz", 1.73, "emma", 1.68, "mom" , 1.71, "dad" , 1.89]
    # the above variable fam has a list with strings and floats
    print(Fam[7])
    # print out the 7th index :1.89

    Fam[7] = 1.80
    # change the element in the 7th index
    # replace the last number 1.89 with the number 1.80

    print(fam)
    # print out the fam list, with the 7th index changed:
    [ "liz", 1.73, "emma", 1.68, "mom" , 1.71, "dad" , 1.80]
    # output

    fam[0:2] = ["lisa”, 1.74]
    # change the elements in the index 0 and 1
    # the element in Index 0 will be Lisa
    # the element in index 1 will be 1.74
    # the 2nd index is not changed because according to the format it is supposed to be excluded

    print(fam)
    # print out the fam list, with index 0 and 1 changed
    [ "lisa”, 1.74, "emma", 1.68, "mom" , 1.71, "dad" , 1.80]
    # output

    How to add and remove elements ?

    In the below example we are going to add elements to, and remove elements from, a list:

    fam = [ "liz", 1.73, "emma", 1.68, "mom" , 1.71, "dad" , 1.89]
    # the above variable fam has a list with strings and floats

    fam + ["me", 1.79]

    print(fam)

    [ "liz", 1.73, "emma", 1.68, "mom" , 1.71, "dad" , 1.89, "me", 1.79 ]

    del(fam[2])

    How to copy elements in a list into another list, and only make changes to the 2nd list ?

    In the below example we will copy the items from one list, into another list. Then replace one of the elements in the second list.

    x = ["a", "b", "c"]
    # the variable x contains a list
    y = list(x)
    # the variable y contains a copy of the list x
    y = x[:]
    # the variable y contains the list inside the variable x
    y[1] = "z"
    # the element in the index 1 of only the list y will be changed to “z”

    print(x)
    # print elements stored in x
    x: ["a", "b", "c"]
    # output; Notice that the item in index 1 maintained its original value.
    Print(y)
    # print elements stored in y
    y: ["a", “z”, "c"]
    # output ; notice that the element in index 1 has the new changed value.

    Overview of numpy package

    The Numpy package makes the process of making calculations on arrays a lot smoother and faster. Numpy arrays can contain only one type. If the list contains a float, a string and a boolean, the resulting array will just have strings. The float and boolean will be converted to strings.

    How to convert a list into a numpy array ?

    height = [1.73, 1.68, 1.71, 1.89, 1.79]
    # The constant height stores this list

    Import numpy as np
    # import the numpy package as np
    np_height = np.array(height)
    # The constant np_height stores a numpy array containing the items in the constant height
    print(np_height)
    # print the numpy array

    ([1.73, 1.68, 1.71, 1.89, 1.79])
    # this is the output; the smooth brackets indicates that this is a numpy array.

    How to print out the mean of a numpy array ?

    Mean is identified by adding up all the numbers, and then dividing the sum by the number of numbers.

    import numpy as np
    # import the numpy package as np
    arr = [20, 2, 7, 1, 34]
    # the array and its values
    np_arr = np.array(arr)
    # convert the regular array to a numpy array

    print(np.mean(np_arr))
    # print out the mean of the numpy array
    12.8
    # output

    Math for calculating the mean:
    20 + 2 + 7 + 1+ 34 = 64
    64 / 5 = 12.8

    How to print out the median of a numpy array ?

    Median is the middle value in a list of numbers. To find it, you first have to re-order the numbers from smallest to largest.

    import numpy as np
    # import the numpy package as np
    arr = [20, 2, 7, 1, 34]
    # the array and its values
    np_arr = np.array(arr)
    # convert the regular array to a numpy array

    print(np.median(np_arr))
    # print out the median of the numpy array
    7.0
    # output

    Math for calculating the median:
    First reorder them from lowest to highest: 1, 2, 7, 20, 34 .
    There are 5 numbers and the middle number is 7.

    How to print out the median of a 2d numpy array ?

    Median is the middle value in a list of numbers. To find it, you first have to re-order the numbers from smallest to largest.

    import numpy as np
    # import the numpy package as np
    np_2d = np.array([
    [1.73, 1.68, 1.71, 1.89, 1.79],
    [65.4, 59.2, 63.6, 88.4, 68.7]
    ]) # write in our 2d numpy array

    print(np.median(np_2d))
    # print out the median
    30.545
    # output

    Math to calculate the median:
    First reorder them from lowest to highest:
    [1.68, 1.71, 1.73, 1.79, 1.89, 59.2, 63.6, 65.4, 68.7, 88.4]
    There are a total of 10 numbers.
    In this example, there is no "middle" number, because there are an even amount of numbers.
    Because of this, the median of the list will be the mean (that is, the usual average) of the middle two values within the list.
    1.89 + 59.2 = 61.09
    61.09 / 2 = 30.545

    How to printout out the ‘correlation coefficient matrix’ of a 2d numpy array ?

    Here we are calculating the relationship between different variables in the data set. For example: Below we have a data set for three peoples height and weight. A row contains one persons data (x, y) = (height, weight). So the first person has a height of 83 inches and weights 120 pounds.

    bmi = np.array([
    [83,120],
    [84,166],
    [96,140],
    ])

    We want to find the correlation coefficient between the height and the weight. To do this, we can use the python numpy correlation formula.

    print(np.corrcoef(bmi[:,0], bmi[:,1]))
    # print out the correlation
    # between the first column and the second column
    [[ 1. -0.00599315]
    [-0.00599315 1. ]]
    # output
