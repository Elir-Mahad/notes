How to connect your github account to your computer

1. Identify your github username and user email:

  • Your user name is in your github profile.
  • Your email is in your github settings.
  • Go to the github settings.
  • On the left panel, click on the Emails tab.

    • 2. Declare globally your github username and email:

    Make sure you have github in your computer by checking your github version.

    git --version

    Response in terminal should be:

    git version 2.27.0

    Declare your user name and email:

    git config --global user.name 'mona-lisa'
    git config --global user.email 'mona-lisa@gmail.com'

    3. Check that correct user information is in vsc:

    git config --list

  • Response in the terminal should be:

    • user.name=mona-lisa
    user.email=mona-lisa@gmail.com
